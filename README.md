# Números Ganadores RD - Sitio legal para GitHub Pages

Proyecto listo para publicar en GitHub Pages con tres páginas requeridas para Meta/Facebook:

- `privacy-policy.html`
- `data-deletion.html`
- `terms-of-service.html`

## Publicación rápida

1. Crea un repositorio público llamado `numeros-ganadores-rd-legal`.
2. Sube todos los archivos de este proyecto.
3. En GitHub, ve a **Settings > Pages**.
4. En **Build and deployment**, selecciona **Deploy from a branch**.
5. Elige la rama `main` y la carpeta `/root`.
6. Guarda los cambios y espera la publicación.

## URLs esperadas

Reemplaza `TU-USUARIO` por tu usuario de GitHub:

- `https://TU-USUARIO.github.io/numeros-ganadores-rd-legal/privacy-policy.html`
- `https://TU-USUARIO.github.io/numeros-ganadores-rd-legal/data-deletion.html`
- `https://TU-USUARIO.github.io/numeros-ganadores-rd-legal/terms-of-service.html`

## Personalización opcional

- Cambia las fechas si luego haces una actualización.
- Ajusta el texto legal según tus operaciones reales.
- Si usas dominio propio, estas páginas también pueden servirse allí.
